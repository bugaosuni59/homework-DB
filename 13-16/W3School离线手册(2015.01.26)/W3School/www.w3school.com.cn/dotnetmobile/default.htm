.NET Mobile 教程

  • Next Page

在本教程中，你将学习移动 .NET（.NET Mobile）。

本教程的主要内容是：如何利用 .NET 框架的扩展来创建移动应用程序，这种扩展被称为微软移动因特网工具包（MMIT）或者简易 .NET Mobile。

开始学习 .NET Mobile ！

内容目录

.NET Mobile 简介
本章阐述 ASP.NET Mobile 的概念，以及如何被使用。
.NET Mobile 实例
这个简单的例子演示出 ASP.NET Mobile 的力量所在。
.NET Mobile Emulators
本章介绍一些用来测试移动应用程序的模拟器。
.NET Mobile Forms
本章向你解释移动表单的使用。
.NET Mobile 事件
本章介绍移动表单中事件的使用。
.NET Mobile 输入
本章介绍移动输入控件的使用。
.NET Mobile 验证
本章介绍移动输入验证的使用。
.NET Mobile Lists
本章介绍移动列表控件的使用。
.NET Mobile Selection
本章介绍移动选项列表控件的使用。
.NET Mobile Images
本章介绍移动图像控件的使用。
.NET Mobile Utilities
本文讲解 mobile utility 控件的使用，比如 AdRotator 和 PhoneCall 控件。
.NET Mobile 参考手册
完整的移动控件对象的参考索引，包括属性、方法和事件。

本章列出了 1.0 版本支持的移动设备。

本章列出了对移动设备进行支持的更新。
